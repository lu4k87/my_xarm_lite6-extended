Commands: rqt Tools (GUI)

rqt ist das offizielle GUI-Framework von ROS 2. Es bietet interaktive Fenster, um den Systemzustand visuell anzuzeigen oder Nachrichten zu senden, als würdest du eine normale Desktop-Applikation bedienen.

Visuelle Netzwerk-Karte

rqt_graph
Öffnet ein Fenster, das jeden laufenden Node als Blase darstellt und alle Datenströme (Topics) als verbundene Pfeile dazwischen zeichnet. Perfekt zum Debuggen von Abhängigkeiten.

Service-Tester

ros2 run rqt_service_caller rqt_service_caller
Ein Pop-up, mit dem du Parameter bequem wie in einem Formular ausfüllen und ROS 2 Services live am Roboter auslösen kannst (z.B. "Fahre zu Scanner-Pose"). Du ersparst dir das Tippen langer Terminal-Befehle.

Log-Zentrale

ros2 run rqt_console rqt_console
Sammelt alle Meldungen, Warnungen und Fehler aller Nodes an einem Ort. Man kann filtern, sortieren und übersichtlich lesen, ohne hunderte Terminal-Fenster checken zu müssen.
