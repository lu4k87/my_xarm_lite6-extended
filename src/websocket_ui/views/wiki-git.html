Wiki: Git Online Repository

Lokales Git mit Online-Server verbinden

Um das lokale Git-Repository online zu sichern (z.B. auf GitHub oder GitLab) und KI-Agenten Updates bzw. Sicherungen machen zu lassen, muss es mit einem "Remote-Repository" (einem Online-Speicher) verknüpft werden.

Schritt 1: Ein leeres Repository online erstellen

Erstelle auf GitHub ein neues Repository ohne README, .gitignore oder Lizenz. Es muss komplett leer sein.

Schritt 2: Die Online-URL kopieren

Kopiere die HTTPS oder SSH URL des neuen Repositories (z.B. https://github.com/DeinName/dein-repo.git).

Schritt 3: Das lokale Git verknüpfen & hochladen

# Den Online-Server als "origin" hinzufügen
git remote add origin <DEINE_URL>

# Den aktuellen Branch auf "main" umbenennen (falls "master")
git branch -M main

# Die Daten hochladen und origin als Standard festlegen
git push -u origin main

So arbeitet der KI-Assistent damit (Antigravity):

Sobald dieser erste Push funktioniert hat, kann der Assistent den Code verwalten. Du kannst einfach sagen:

  • "Speichere die heutigen Änderungen und lade sie auf Git hoch"
  • "Lade den neuesten Code von GitHub herunter"
  • "Checke den aktuellen Git Status"
  • "Welche Commits wurden heute gemacht?"
