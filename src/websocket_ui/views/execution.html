Execution Übersicht: lite6.sh

Übersicht aller Hintergrundprozesse und Nodes, die für den Betrieb des Roboter-Systems und des Dashboards initialisiert werden.

Kinematik & Steuerung

X-Arm MoveIt Servo

ros2 launch xarm_moveit_servo lite6_moveit_servo_realmove.launch.py
robot_ip:=192.168.1.175
add_vacuum_gripper:=True
report_type:=dev
  • Baut TCP/IP Verbindung zur Lite6 Hardware auf.
  • Aktiviert MoveIt Servoing.

Motion Sequence

ros2 launch motion_sequence motion_sequence_launch.py
  • Verwaltet geplante Bewegungsabläufe.

Move To Coordinator

ros2 run move_to_coordinator move_to_coordinator
  • Handler für dynamische Zielkoordinaten.

Vision & Sicherheit

YOLO Homographie

ros2 run yolo_object_detector yolo_homography_node
  • Objekterkennung (YOLO) & 3D-Koordinaten.

Collision Check

ros2 run collision_check checker
  • Echtzeit-Sicherheitsüberwachung.

RViz Publisher

ros2 run rviz_marker marker_publisher
  • Generiert 3D-Marker für RViz.

Sprachsteuerung

Whisper Bringup

ros2 launch whisper_bringup bringup.launch.py
silero_vad_use_cuda:=True
  • Startet KI-Spracherkennung.

Whisper Stream Demo

ros2 run whisper_demos whisper_on_key
  • Aktiviert Mikrofon bei Tastendruck.

Voice Command Listener

ros2 run voice_command_listener listener
  • Parst Text in Roboterkommandos.

Dashboard & System

ROS Bridge Websocket

ros2 launch rosbridge_server rosbridge_websocket_launch.xml
  • Schnittstelle zw. ROS (DDS) und Web (JSON).

Workspace Analyzer

python3 workspace_analyzer.py
  • Backend für das Node Dashboard.
