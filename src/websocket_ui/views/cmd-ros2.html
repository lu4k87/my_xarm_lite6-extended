Commands: ROS 2 CLI

Die wichtigsten Befehle zur Interaktion mit dem ROS 2 System, Nodes und Topics.

Nodes & Info

ros2 node list
Listet alle aktuell laufenden Nodes im System auf.
ros2 node info /node_name
Zeigt Details (Publisher, Subscriber, Services) einer bestimmten Node an.
ros2 run package_name executable_name
Startet eine einzelne kompilierte C++ Node oder ein Python-Skript.

Topics & Messages

ros2 topic list -t
Listet alle aktiven Topics inklusive ihres Message-Typs (z.B. std_msgs/msg/String) auf.
ros2 topic echo /topic_name
Druckt die Live-Nachrichten, die aktuell über ein Topic gesendet werden, ins Terminal.
ros2 topic hz /topic_name
Misst die Frequenz (Hz), mit der Nachrichten auf einem Topic veröffentlicht werden.
ros2 interface show msg_type
Zeigt die innere Struktur (Felder und Datentypen) einer Message an (z.B. geometry_msgs/msg/Pose).
