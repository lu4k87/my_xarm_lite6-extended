Commands: Git Versioning

Befehle zum Speichern und Verwalten von Code-Versionen.

git status
Zeigt an, welche Dateien geändert, gelöscht oder neu hinzugefügt wurden.
git add .
Fügt alle Änderungen zum nächsten Commit ("Speicherung") hinzu.
git commit -m "Message"
Speichert die zuvor hinzugefügten Dateien lokal als festen Meilenstein (Commit) ab.
git push origin branch
Lädt die lokalen Commits auf den Online-Server (z.B. GitHub) hoch.
