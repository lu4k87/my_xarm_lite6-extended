Commands: Colcon Build

Befehle zum Kompilieren und Erstellen von ROS 2 Workspaces.

Kompiliert den gesamten Workspace. Sehr wichtig (Python): Erlaubt das Ändern von Python-Skripten, ohne danach neu kompilieren zu müssen.
colcon build --symlink-install
Kompiliert nur ein spezifisches Paket, anstatt des gesamten Workspaces (Sammelt massiv Zeit bei C++).
colcon build --packages-select pkg_name
Macht neu kompilierte Nodes oder neue Packages im aktuellen Terminal verfügbar. Muss nach jedem Start eines neuen Terminals (oder per .bashrc) ausgeführt werden.
source install/setup.bash
