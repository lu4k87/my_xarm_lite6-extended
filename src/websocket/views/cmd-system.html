Commands: Linux System

Allgemeine hilfreiche Ubuntu / Linux Terminal Befehle.

htop
Zeigt CPU-Auslastung, RAM-Verbrauch und alle aktuell laufenden Prozesse in Echtzeit. (Mit `q` beenden)
killall -9 process_name
Erzwingt das sofortige Beenden aller Prozesse mit diesem Namen (hilft, wenn sich ROS mal aufgehängt hat).
nano ~/.bashrc
Öffnet die globale Konfigurationsdatei deines Terminals im simplen Editor.
chmod +x script.py
Macht eine Skript-Datei ausführbar (wichtig für direktes Ausführen ohne "python3 davor").
